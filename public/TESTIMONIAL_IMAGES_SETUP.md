# 🌟 好评头像设置指南

## 📸 如何添加好评头像图片

### 步骤1：准备图片文件
1. **图片规格建议**：
   - 尺寸：150x150 像素（正方形）
   - 格式：JPG 或 PNG
   - 文件大小：建议控制在 50KB 以内
   - 背景：建议使用专业头像或半身照
   - 风格：正式、专业、有亲和力

### 步骤2：命名和放置
将图片文件放在以下位置：
```
compass/
└── public/
    └── images/
        └── testimonials/
            ├── sarah-johnson.jpg
            ├── michael-chen.jpg
            └── emma-rodriguez.jpg
```

### 步骤3：图片内容建议
为了完美配合网站的专业形象，建议头像包含：
- ✨ 专业的商务头像
- 👔 正式或半正式的服装
- 😊 友好、自信的表情
- 🎯 清晰的正面或侧面照
- 🚫 避免过于休闲或模糊的照片
